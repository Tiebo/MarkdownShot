# MarkShot

## 开发原因

今天需要复制一段很长的markdown然后截图，但是发现markdown是在太长了，很难截图完整，于是搓了一个这样的简易网站

## 主要特性

- 支持数学公式

- 支持导出图片宽度自定义

- 支持代码高亮

- 支持自定义背景颜色
